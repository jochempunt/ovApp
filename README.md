# Departures

Real-time Dutch public transport departure board. Built because I got tired of slow official apps when I just needed to check if my bus was coming.

**Live site:** [ov-app-ten.vercel.app](https://ov-app-ten.vercel.app/)

## Features
- Live departure data from OV-API (buses, trams, metros)
- Search any bus or metro stop in the Netherlands
- Multiple custom themes
- PWA supportâ€”install to home screen for one-tap access
- Shareable links with stop encoded in URL

## Built with
- React + TypeScript
- Vite
- Material-UI (MUI)
- TanStack Query for data fetching and caching
- OV-API
